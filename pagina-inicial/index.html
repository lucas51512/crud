O que é esse sistema

Esse projeto foi feito para a aula de linguagem de programação, ministrada pelo professor Almir Rogério Camolési, realizada na Universidade: Fundação Educacional do Município de Assis - FEMA, usando de linguagem Javascript pura, bootstrap, CSS3 e HTML5.

O trabalho foi realizado pelo aluno Lucas Barbosa Brancalhão nos dias 20 a 27 de Junho, utilizando das ferramentas Trello, para a organização, e GitHub para que o mesmo fosse guardado em um repositório.

Link do repositório Git: https://github.com/lucas51512/crud.git

Link da ista do Trello: https://trello.com/b/g0lK0Y0I/trabalho-almir

Link da video-aula usada para fazer o CRUD: https://www.youtube.com/watch?v=_HEIqE_qqbQ&ab_channel=FernandoLeonid

Link da video-aula usada para fazer o redirecionamento da pagina no login: https://www.youtube.com/watch?v=c-kUoHk9ggw&ab_channel=DevMedia

A autenticação do login foi feita manualmente com o auxílio da video-aula para redrecionamento de paginas.
